# React SSR Starter Kit

**Features:**
* Webpack 4
* Babel 7
* React
* React-router
* TypeScript
* ESlint
* Stylus
* CSS Modules
* PostCSS
* Express

## Usage

```bash
npm start          # build development and start node server with SSR
npm run dev        # start webpack-dev-server
npm run build:dev  # build development
npm run build      # build production
```
